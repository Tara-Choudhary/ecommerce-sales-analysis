# 📊 E-commerce Sales Analysis using SQL

## 📌 Project Overview
This project focuses on analyzing an E-commerce sales dataset using SQL to extract meaningful business insights.  
The goal is to understand sales performance, profitability, and regional trends by writing efficient SQL queries.



## 🎯 Objectives
- Analyze overall sales and profit performance
- Identify top-performing product categories
- Determine the most profitable regions
- Detect loss-making transactions
- Build strong SQL and data analysis skills



## 🛠️ Tools & Technologies
- SQL (MySQL)
- DBeaver (Database Client)
- CSV Dataset



## 📂 Dataset
The dataset used in this project is an artificial E-commerce sales dataset containing:
- Order Date
- Product Name
- Category
- Region
- Sales
- Quantity
- Profit

## 🔍 Analysis Performed

### 1️⃣ Data Exploration
- Total number of orders
- Time range of the dataset
- Unique categories and regions
- Total sales, profit, and quantity

### 2️⃣ Business Insights
- Highest revenue-generating category
- Most profitable region
- Top 5 selling products

### 3️⃣ Advanced Analysis
- Monthly sales trends
- Profit margin by category
- Identification of loss-making orders



## 📈 Key Insights
- Certain categories contribute significantly to overall revenue
- Some regions generate higher profits than others
- Loss-making orders highlight potential business issues
- Monthly trends help understand seasonal sales patterns



## 🚀 Skills Demonstrated
- SQL Aggregations (SUM, COUNT)
- GROUP BY and ORDER BY
- Data exploration techniques
- Business-oriented query writing
- Problem-solving and analytical thinking

## ▶️ How to Run This Project
1. Create a database:
   ```sql
   CREATE DATABASE ecommerce_analysis;
   USE ecommerce_analysis```
2. Import the dataset (`ecommerce_sales.csv`) into your SQL database
3. Run SQL scripts from the sql/ folder in order:
   - 01_database_setup.sql
   - 02_exploration.sql
   - 03_business_queries.sql
   - 04_advanced_analysis.sql
